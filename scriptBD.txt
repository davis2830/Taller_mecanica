Cuando la base este creada y utilizable desde el proyecto ejecutar los siguientes comandos en la ruta Taller-mecanica/taller_mecanico

python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser

estos comandos realizaran las migraciones correspondientes y crearan las tablas en la base de datos.


Tener en cuenta que antes de esto tiene que tener el entorno virtual activado e instalado el archivo requirements.txt que contiene todas las librerías para funcionar el proyecto.
